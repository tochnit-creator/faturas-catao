Guia de atualização mensal
Tudo o que você precisa mandar para o Claude
para manter seu histórico de faturas sempre atualizado
✅ Checklist mensal
  • Baixei o PDF da fatura fechada no app Bradesco (quando fechar o ciclo)
  • Baixei o PDF do extrato em aberto para ver lançamentos recentes
  • Tenho o arquivo faturas_bradesco.html atual (do GitHub)
  • Abri uma nova conversa com o Claude em claude.ai
  • Mandei os 3 arquivos juntos com a mensagem de atualização
  • Fiz o upload do HTML novo no GitHub substituindo o antigo
1
Baixe os PDFs no app Bradesco
Faça isso após o fechamento da fatura (geralmente dia 28)

No app Bradesco acesse Cartões → Fatura → Ver fatura completa e salve o PDF. Depois vá em Extrato em aberto e salve também.

Fatura fechada ex: Bradesco_Fatura_Jun2026.pdf
Extrato aberto ex: Bradesco_ExtratoFaturaAberta_28-06-2026.pdf

💡 Baixe sempre os dois: a fatura fechada tem os dados definitivos, o extrato aberto mostra o que já acumulou para o próximo ciclo.
2
Pegue o arquivo HTML atual do GitHub
É a memória do sistema — sem ele o histórico se perde

Baixe o arquivo faturas_bradesco.html do seu repositório GitHub. Ele contém todos os meses anteriores, parcelas e planejamento.

⚠️ Sempre pegue a versão mais recente do GitHub, nunca uma versão antiga salva no computador.
3
Abra o Claude e mande os 3 arquivos
Nova conversa + anexos + mensagem pronta abaixo

Acesse claude.ai, inicie uma nova conversa, anexe os 3 arquivos e use esta mensagem:

Oi! Segue a fatura fechada de [MÊS/ANO], o extrato em aberto de [DATA] e o HTML atual. Por favor: 1. Adicione a aba do novo mês com os lançamentos 2. Atualize a aba "Em aberto" com os novos lançamentos 3. Recalcule o planejamento com as parcelas atualizadas 4. Mantenha o histórico dos meses anteriores Meu salário continua R$ 9.000/mês.
4
Suba o HTML novo no GitHub
Substitua o arquivo antigo pelo novo — leva 1 minuto

① Acesse seu repositório faturas-catao no GitHub
② Clique no arquivo faturas_bradesco.html
③ Clique no ícone de lápis ✏️ → Upload file
④ Arraste o arquivo novo → Commit changes

✅ Em 1-2 minutos o GitHub Pages atualiza e o link já mostra o histórico completo.
📅 Quando fazer cada coisa
Todo dia 28
Fatura fecha. Baixe o PDF da fatura fechada e deixe separado.
Todo dia 10
Vencimento. Pague o total — nunca o mínimo!
Qualquer dia
Baixe o extrato em aberto para ver lançamentos recentes.
Uma vez por mês (sugestão: dia 1)
Mande os 3 arquivos para o Claude e atualize o GitHub. Menos de 5 minutos.
💬 Dicas importantes
  • Nunca pague o mínimo. Juros chegam a 16,99% a.m — a dívida dobra em meses.
  • Junho é crítico. 57% do salário vai em parcelas. Evite novas compras parceladas até agosto.
  • A partir de novembro as parcelas caem muito. Momento de guardar dinheiro.
  • Sempre mande o HTML atual — sem ele o histórico se perde.
  • Não precisa ser perfeito. Se esquecer um mês, manda depois que o Claude reconstrói.
